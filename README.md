# Housing Market Analysis Dashboard 

## Overview
This Power BI dashboard analyzes U.S. housing affordability by examining the relationship between household income, home prices, and price per square foot. The goal is to identify affordability trends across cities and highlight regional cost differences.
--
## Key Insights
- Higher income cities tend to have significantly higher home prices, indicating a strong positive correlation between income and housing costs.
- Price per square foot varies widely by state, highlighting regional disparities in housing markets.
- Some cities exhibit lower affordability despite moderate incomes, suggesting cost-of-living imbalances.
